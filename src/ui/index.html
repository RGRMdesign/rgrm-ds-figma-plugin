Layer mapping

Name layers in your template component to fill them automatically:

  • name — variable name
  • value — fully resolved value
  • alias / reference — referenced variable (empty if direct value)
  • alias-collection / reference-collection — collection of referenced variable (empty if direct value)
  • type — STRING, FLOAT, BOOLEAN, or COLOR
  • description — description
  • collection — source collection name
  • color / swatch — color swatch (COLOR)
  • bind-color — binds COLOR variable to fill
Select a component, component set, or instance on the canvas

Export all variable collections

Downloads a figma.zip with one directory per collection. Single-mode collections export as tokens.json; multi-mode collections export as [mode].tokens.json per mode.
